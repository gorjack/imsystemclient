 ## 软件架构说明

软件整体采用C/S 架构。


![输入图片说明](https://images.gitee.com/uploads/images/2021/0503/123258_c1ac8c98_7500763.png "im.png")


## 编译安装

直接打开sln 编译就可以，目前只配置了x86 debug的, ps(用vs2017编译)



## 基本实现功能

- 登录功能模块
- 主界面功能模块
- 添加好友 删除好友等
- 托盘功能
